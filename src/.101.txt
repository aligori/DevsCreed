<!------ Create project -------->

Create laravel project:
composer create-project laravel/laravel example-app 7.28.0
OR
composer global require laravel/installer
laravel new example-app


<!------ Start project -------->
// remember to be in the src folder
// if not then: cd src

After cloning the repo:
composer update
composer install
	OPTIONAL TAG (not recommended)
	--ignore-platform-reqs

Start the localhost:
php artisan serve

Update the cache (after changes to .env):
php artisan config:cache


<!---------- Views ----------->

Setup Bootstrap & Vue UI:
php artisan ui vue --auth
npm install
npm run dev

Compile views to public folder:
npm run dev

Install Fontawesome:
npm install --save @fortawesome/fontawesome-free


<!-------- Database ------->

Generate a migration, seeder, factory, and resource controller for the model:
php artisan make:model <model_name> --all

Generate migrations for the given models:
php artisan migrate

Get default/initial/dummy table values:
php artisan db:seed


<!------- List CRUD functions ------->

Route CRUD functionalities:
php artisan route:list --name=<name>


<!------ Troubleshooting ------>

php.ini Dynamic Extensions:
extension=php_fileinfo.dll

Generate application key:
php artisan key:generate


<!------- Comments --------->
